# Game Design Document (GDD)

## 1. Структура рана (Run Structure)

### Карта рана
Ран состоит из **5–7 сражений** с нарастающей сложностью:

```
[Старт]
   ↓
[Этаж 1] Новичок — знает 5 ходов теории
   ↓
[Этаж 2] Любитель — знает 8 ходов, вариативен
   ↓
[Этаж 3] Выбор пути (два противника, разные дебюты)
   ↓
[Этаж 4] Ветеран — знает 12 ходов
   ↓
[Этаж 5] БОСС — знает 15+ ходов, высокий Stockfish
   ↓
[Победа / Новый ран]
```

### Рандомизация
- Противники на каждом этаже выбираются из пула (зависит от открытых персонажей)
- Порядок противников частично случаен, боссы всегда финальные
- Стартовый дебют игрока выбирается перед раном

---

## 2. Система здоровья

| Событие | Эффект |
|---------|--------|
| Победа в партии | +1 HP (макс. 3) |
| Поражение | −1 HP |
| HP = 0 | Game Over |
| Ничья | HP не меняется, противник считается «сбежавшим» |

**Стартовое HP:** 3

### Альтернативные режимы (планы)
- **Хардкор**: 1 HP, сразу Game Over
- **Марафон**: бесконечные противники, считаются очки

## 3. Противники (Enemy System)

### Структура противника

```yaml
enemy:
  id: "sicilian_dragon"
  name: "Дракон"
  title: "Защитник Сицилии"
  portrait: "dragon.svg"
  color: "#8B0000"          # акцентный цвет
  plays_as: "black"         # за какой цвет играет
  opening: "Сицилианская защита, вариант Дракона"
  moves: ["e4", "c5", "Nf3", "d6", "d4", "cxd4", ...]
  theory_depth: 12          # знает теорию до хода 12
  stockfish_elo: 1400       # сила вне теории
  quotes:
    intro: "Никто не выходил из этой норы живым."
    losing: "Как ты посмел нарушить порядок ходов?!"
    winning: "Твоя теория слаба, как пешка на h7."
    draw: "Уходи. Ты недостоин моего времени."
```

### Пул противников (первый релиз, v0.1)

| # | Имя | Дебют | Цвет | Сложность |
|---|-----|-------|------|-----------|
| 1 | Итальянец | Итальянская партия | Белые | Новичок |
| 2 | Испанец | Испанская партия | Белые | Любитель |
| 3 | Сицилийка | Сицилианская (Najdorf) | Чёрные | Любитель |
| 4 | Каро | Защита Каро-Канн | Чёрные | Ветеран |
| 5 | Французский Граф | Французская защита | Чёрные | Ветеран |
| 6 | Дракон | Сицилианская (Dragon) | Чёрные | Ветеран |
| 7 | Король Гамбита | Королевский гамбит | Белые | Босс |
| 8 | Призрак Фишера | Защита Нимцовича | Чёрные | Босс |

### Реплики (Quotes)
Каждый противник имеет минимум 4 реплики:
- **intro** — при встрече
- **in_theory** — пока партия идёт по теории
- **out_theory** — когда игрок отклонился
- **winning** — когда противник побеждает по позиции
- **losing** — когда игрок побеждает по позиции
- **checkmate_win** — мат поставлен противником
- **checkmate_loss** — мат поставлен игроком
- **draw** — при ничье

## 4. Система дебютов (Opening System)

### Opening Book
- Хранится как JSON-дерево ходов (PGN-формат или кастомный)
- Источник: Lichess opening database (открытая, бесплатная)
- Для каждого противника — своя ветка Opening Book

### Индикатор теории
На доске отображается:
- Зелёная рамка хода: ты в теории
- Жёлтая: спорный ход (есть в книге, но редкий)
- Красная: выход из теории
- Иконка книги с номером хода: глубина текущей позиции

### Opening Cards
Карта дебюта содержит:
- Название варианта
- Главная линия (первые 10–15 ходов)
- 1–2 ключевых идеи (текст)
- Диаграмма позиции на ключевом ходу

Карты открываются за:
- Победу над конкретным противником
- Определённое количество теоретических ходов в ране
- «Идеальный» дебют (все ходы по книге)

## 5. Stockfish Integration

### Уровни силы
| Уровень | ELO | Используется для |
|---------|-----|-----------------|
| 1 | ~800 | Обучающий режим |
| 2 | ~1000 | Новички |
| 3 | ~1200 | Любители |
| 4 | ~1400 | Ветераны |
| 5 | ~1600 | Боссы (1-й ран) |
| 6 | ~1800 | Боссы (поздние раны) |

### Поведение ИИ
- **В теории**: ИИ ходит по своей Opening Book (детерминировано)
- **Вне теории**: ИИ использует Stockfish с ограничением ELO
- **Переход**: когда теория кончается, есть пауза (~1 сек) — визуальный сигнал

### Анализ после партии
- Показывается граф оценки позиции по ходам
- Отмечаются «зевки» (blunders) игрока
- Подсказка: на каком ходу игрок вышел из теории

```
┌─────────────────────────────────────────────┐
│  [Портрет врага]   [Имя]   [HP врага?]       │
│  "Реплика противника здесь..."               │
├──────────────┬──────────────────────────────┤
│              │   [Шахматная доска]           │
│  [Opening    │                               │
│   Cards]     │                               │
│              │                               │
│  [Knowledge  │                               │
│   Bar ████░] │                               │
├──────────────┴──────────────────────────────┤
│  Ход 7 / Теория: ✓  Глубина: e4 e5 Nf3 Nc6  │
│  [Сдаться]                      [HP: ❤❤❤]   │

```

### Экран между раундами
- Показ Opening Card если разблокирована
- Выбор пути (если развилка)
- Счёт знания за раунд

### Экран Game Over
- Сколько противников побеждено
- Какой дебют противника «убил» игрока
- Подсказка по этому дебюту
- [Новый ран] [Изучить дебют]

## 7. Прогрессия между ранами (Meta-progression)

- **Разблокированные противники** сохраняются
- **Opening Cards** сохраняются в «библиотеку»
- **Статистика**: любимый дебют, процент теоретических ходов
- **Достижения**: «Сыграл идеально Испанскую», «Победил Дракона» и т.д.

Прогресс сохраняется в localStorage (без бэкенда на первом этапе).
