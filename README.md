# Opening Gauntlet

Браузерная шахматная рогалик-игра для изучения дебютов.

## Что это

Ты сражаешься с чередой противников-персонажей. Каждый играет строго определённый дебют. Знаешь теорию — продвигаешься дальше. Ошибаешься в дебюте — противник усиливается.

## Быстрый старт разработки

```bash
# Phase 0: Foundation
npm create vite@latest . -- --template vanilla-ts
npm install
npm install chess.js
# Добавить stockfish.wasm в public/stockfish/
npm run dev
```
